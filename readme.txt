Just another search refinement plugin depends on custom filed.

== Description ==

The searching is narrow down a search depends on custom field you saved.
So, it's basically avoid a not found.
You'd like to use in widgets.


== Installation ==

1. Install automatically through the Plugins menu and Add New button (or upload the entire search-refinement folder to the /wp-content/plugins/ directory)
2. Activate the plugin
3. Go to Appearance -> Widgets
4. Drag and drop the Search Refinement Widget to the desired sidebar
5. select items which custom field keys.
